1. Sum

Given an array of integers, return the sum of all elements.

Examples

Input:
nums = [2,7,11,15]

Output:
35

Explanation: 2 + 7 + 11 + 15 = 35

Input:
nums = [3,2,4]

Output:
9

Input:
nums = [3,3]

Output:
6

Constraints

  • 1 <= nums.length <= 10^4
  • -10^9 <= nums[i] <= 10^9

Concepts Needed

  • Basic array traversal
  • Summation
