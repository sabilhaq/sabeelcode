5. Check Prime

Given an integer n, return true if it is a prime number, and false otherwise.

A prime number is a number greater than 1 that has no divisors other than 1 and itself.

Examples

Input:
n = 2

Output:
true

Input:
n = 4

Output:
false

Explanation: 4 is divisible by 2.

Input:
n = 17

Output:
true

Constraints

  • 1 <= n <= 10^6
