6. Factorial

Given an integer n, return the factorial of n.

The factorial of a number n (denoted as n!) is the product of all positive integers less than or equal to n.

Examples

Input:
n = 0

Output:
1

Input:
n = 5

Output:
120

Input:
n = 7

Output:
5040

Constraints

  • 0 <= n <= 20
