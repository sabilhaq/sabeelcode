7. Fibonacci

Given an integer n, return the first n numbers of the Fibonacci sequence.

The Fibonacci sequence is defined as:
F(0) = 0, F(1) = 1, and F(n) = F(n-1) + F(n-2) for n >= 2.

Examples

Input:
n = 5

Output:
[0,1,1,2,3]

Input:
n = 8

Output:
[0,1,1,2,3,5,8,13]

Constraints

  • 0 <= n <= 30
